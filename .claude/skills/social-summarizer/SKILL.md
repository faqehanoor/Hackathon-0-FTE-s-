## Overview
This skill creates performance summaries for social media platforms.

## When to Use
- After posting content to track performance
- Weekly to summarize social media metrics
- When evaluating campaign effectiveness

## Parameters
- platform: facebook, instagram, twitter, or linkedin
- period: Time period to summarize
- include_metrics: Which metrics to include (likes, shares, comments, etc.)

## Process
1. Pull engagement metrics from platform APIs
2. Calculate key performance indicators
3. Compare to historical data
4. Generate insights and recommendations
5. Save summary to /Summaries/ folder
