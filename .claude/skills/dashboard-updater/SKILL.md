---
name: dashboard-updater
description: Updates Dashboard.md with summaries of tasks, status, and recent activity. Use after processing any task.
---

## Overview
Keep the dashboard current for human review.

## When to Use
- After completing a plan or moving file to /Done.
- For daily/weekly summaries.

## Instructions
1. Read current Dashboard.md.
2. Append new section: ## Recent Activity - [Date] Task summary.
3. Update balances or pending if relevant (e.g., from bank watcher).
4. Write back to Dashboard.md without overwriting old content.

## Example
After processing email:
Add to Dashboard:
## Recent Activity
- 2026-02-02 22:55: Invoice request processed. Plan created.

## Checklist
- Use ISO date format.
- Keep updates concise (under 50 words).
- Ensure no duplicates.
