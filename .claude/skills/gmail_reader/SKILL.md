## Overview
This skill handles Gmail message processing and categorization.

## When to Use
- Monitoring inbox for new emails
- Categorizing emails by priority
- Extracting key information from messages
- Identifying action items in emails

## Processing Steps
1. **Read Message**: Access email content, subject, sender
2. **Categorize**: Determine priority (urgent, normal, promotional)
3. **Extract Info**: Identify key details (dates, amounts, requests)
4. **Action Plan**: Determine next steps based on content

## Priority Levels
- **Urgent**: Time-sensitive, requires immediate attention
- **Normal**: Standard business communication
- **Promotional**: Marketing materials, newsletters

## Integration
- Pass results to task planner for appropriate action
- Update dashboard with email status
- Create approval requests for sensitive communications
- Flag important emails for human attention
