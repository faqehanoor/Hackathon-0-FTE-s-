## Overview
This skill generates engaging LinkedIn posts for business promotion and manages the approval process.

## When to Use
- When user wants to promote business on LinkedIn
- When generating sales-generating content
- When creating professional social media presence

## Parameters
- topic: Subject for the LinkedIn post
- audience: Target audience for the post
- tone: Professional, casual, or promotional tone

## Process
1. Generate engaging content with hook, value proposition, and CTA
2. Create hashtags relevant to the topic
3. Draft post in Pending_Approval folder for review
4. After approval, post to LinkedIn via API
5. Log the posting activity
